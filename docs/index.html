Command Line Arguments

Uses

  • PyinstallerHelper.exe Run normally with full interaction and UI
  • PyinstallerHelper.exe <file> Run normally with full interaction and UI, loading a file
  • PyinstallerHelper.exe <file> -a Build a file automatedly, showing the progress window but with no interaction required.
